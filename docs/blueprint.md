# **App Name**: Rate Record

## Core Features:

- Add Product: Allow users to add new products with details like name, unit, GST, party name, page number, bill date and category.
- Record Rate: Enable users to record the rate of a product over time.
- Product Rate History: Display a history of rates for each product.
- Filtering and Sorting: Implement filtering options (newest first, oldest first, A-Z, Z-A) for easy data access.
- Editing product details: Ability to modify and update the entry for products
- Deletion: Ability to remove specific products with all their respective prices
- Summarization: AI powered tool, summarizing rates with trends, outliers and predictions

## Style Guidelines:

- Primary color: Soft Blue (#64B5F6) to evoke a sense of trust and stability.
- Background color: Light Gray (#F0F4F8) for a clean and professional look.
- Accent color: Teal (#4DB6AC) to highlight important actions and information.
- Body and headline font: 'Inter', a sans-serif font for clear readability and a modern feel.
- Use simple, consistent icons for actions like adding, editing, and deleting rates.
- Clear data table layout for easy scanning of product rates and details.
- Subtle transitions when sorting and filtering data to improve user experience.
